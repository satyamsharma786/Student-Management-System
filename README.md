# 📖 Overview

This project is a console-based **Student Management System** developed completely in **Python**.

Instead of creating just another CRUD application, I focused on writing modular code and implementing features that make the project more practical and closer to a real-world management system.

It helped me strengthen my understanding of:

- Functions
- Lists & Dictionaries
- File Handling
- Exception Handling
- Input Validation
- Authentication
- Modular Programming

## Student Login

Students login using

- Roll Number
- Auto Generated Password

Password format

```
studentname + rollnumber
```

Example

```
Rahul Sharma
Roll No : 101

Password

rahulsharma101
```

Whenever the student's name is updated, the password is regenerated automatically.

- Automatic Saving
